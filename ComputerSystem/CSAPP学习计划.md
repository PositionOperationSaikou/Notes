- 身份：后端游戏程序员（希望深入操作系统）
- 当前状态：离职，准备求职
- 每日安排：约2小时刷题 + 3-4小时CSAPP学习

🎯 总体目标
- 系统掌握《深入理解计算机系统》核心概念
- 完成重点章节实验（Bomb Lab、Cache Lab、Shell Lab、Proxy Lab）
- 保持算法练习，兼顾面试准备

🗓️ 详细周计划

第1周
- 章节：第1章 计算机系统漫游、第2章 信息的表示和处理
- 重点：系统概览、数值表示、整数/浮点运算
- 实验：无（可先配置实验环境）
- 刷题：数组、位运算

第2周
- 章节：第3章 程序的机器级表示（上）
- 重点：x86-64汇编基础、控制流、过程调用
- 实验：Bomb Lab（理解汇编与调试）
- 刷题：链表、栈/队列

第3周
- 章节：第3章 程序的机器级表示（下）
- 重点：数据布局、缓冲区溢出
- 实验：Attack Lab（深入理解栈溢出与代码注入）
- 刷题：字符串、哈希表

第4周
- 章节：第4章 处理器体系结构
- 重点：Y86-64指令集、流水线、冒险
- 实验：Architecture Lab（可选）
- 刷题：树、递归

第5周
- 章节：第5章 优化程序性能
- 重点：编译器优化、循环优化、内存性能
- 实验：无（可自己写代码测试优化效果）
- 刷题：动态规划（基础）

第6周
- 章节：第6章 存储器层次结构
- 重点：缓存原理、局部性、缓存友好代码
- 实验：Cache Lab（模拟缓存与优化矩阵转置）
- 刷题：图（基础）

第7周
- 章节：第7章 链接
- 重点：静态链接、符号解析、重定位
- 实验：无（可自己写多文件程序观察符号表）
- 刷题：回溯、贪心

第8周
- 章节：第8章 异常控制流
- 重点：进程、信号、非本地跳转
- 实验：Shell Lab（实现一个简单的 Unix shell）
- 刷题：并发编程题（如生产者-消费者）

第9周
- 章节：第9章 虚拟内存（上）
- 重点：地址翻译、页表、内存映射
- 实验：继续 Shell Lab 或开始 Malloc Lab
- 刷题：设计题（如 LRU 缓存）

第10周
- 章节：第9章 虚拟内存（下）、第10章 系统级I/O
- 重点：动态内存分配、文件I/O、元数据
- 实验：Malloc Lab（实现自己的 malloc/free）
- 刷题：系统设计基础

第11周
- 章节：第11章 网络编程
- 重点：套接字、HTTP、Web服务器
- 实验：Proxy Lab（实现一个并发 Web 代理）
- 刷题：网络相关算法（如最短路径）

第12周
- 章节：第12章 并发编程
- 重点：线程、同步、竞争条件、死锁
- 实验：继续 Proxy Lab（加入并发支持）
- 刷题：多线程编程题

💡 学习建议
1. 主动阅读：每章先快速浏览，再精读重点段落；边读边写笔记
2. 习题必做：课后习题是理解的关键，尤其带 ★ 的题目
3. 实验驱动：实验是 CSAPP 精华，优先完成 Bomb Lab、Cache Lab、Shell Lab、Proxy Lab
4. 结合实践：用 strace、perf 等工具观察系统行为
5. 面试导向：思考“这个问题在面试中怎么问？”

- 晚上 19:00–21:00：实验 / 复习 / 求职准备
